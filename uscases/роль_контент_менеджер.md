# Роль: контент-менеджер

## Назначение роли

Управляет маркетинговым контентом платформы без изменения данных `1С`.

## Границы доступа

- `MVP`: новости, акции, баннеры, обучение, статические страницы.
- `MVP`: без доступа к ценам, остаткам, клиентским документам.
- `Post-MVP`: расширенные редакторские процессы согласования.

## Use cases (MVP)

| ID | Предусловия | Основной поток | Альтернативы/ошибки | Результат | Источник данных | Уведомления | Критерии приемки (QA) |
|---|---|---|---|---|---|---|---|
| CNT-01 | Есть права контент-менеджера | Создает новость -> публикует | Некорректный формат полей | Новость опубликована | Платформа | Опционально внутреннее уведомление | Новость видна на витрине |
| CNT-02 | Есть промо-кампания | Создает/редактирует акцию -> архивирует по сроку | Истек срок акции | Акция отображается/снимается корректно | Платформа | По настройкам | Акция не отображается после архивирования |
| CNT-03 | Нужен новый баннер | Загружает баннер -> настраивает ссылку -> публикует | Ошибка медиаформата | Баннер обновлен | Платформа | Нет | Баннер загружается без деградации верстки |
| CNT-04 | Доступен раздел обучения | Обновляет карточку курса/программы | Неполные поля | Карточка курса актуальна | Платформа | По настройкам | Карточка корректно отображается в разделе обучения |
