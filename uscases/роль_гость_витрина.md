# Роль: гость витрины

## Назначение роли

Неавторизованный пользователь изучает ассортимент и контент до регистрации/входа.

## Границы доступа

- `MVP`: главная, каталог, карточка товара, новости, акции, обучение, о компании.
- `MVP`: поиск только по каталогу/витрине.
- `MVP`: нет доступа к заказам, документам и данным ЛК.
- `Post-MVP`: расширение персонализации витрины (при необходимости).

## Use cases (MVP)

| ID | Предусловия | Основной поток | Альтернативы/ошибки | Результат | Источник данных | Уведомления | Критерии приемки (QA) |
|---|---|---|---|---|---|---|---|
| GST-01 | Открыта витрина | Гость вводит запрос -> получает выдачу каталога -> переходит в карточку товара | Нет результатов -> сообщение и рекомендации | Найден или не найден товар | `1С` (каталог через платформу) | Нет | Поиск работает только по витрине и каталогу |
| GST-02 | Открыта карточка товара | Гость изучает карточку -> нажимает CTA вход/регистрация | Товар недоступен -> показывается состояние недоступности | Переход к авторизации/регистрации | `1С` + контент платформы | Нет | Нет доступа к оформлению заказа без авторизации |
| GST-03 | Открыт раздел контента | Гость читает новость/акцию/обучение | Контент снят с публикации -> не отображается | Информирование клиента | Платформа | Нет | Контент показывается только опубликованный |
