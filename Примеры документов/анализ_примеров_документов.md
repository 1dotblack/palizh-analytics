### 1.3 Наблюдения по номенклатуре

**Форматы артикулов (sku):**

- Стандартный: `ST-10-0,9`, `CE-600-12`, `CS-BK-0,9`
- 1С внутренний: `9999908161`, `9999956937`, `9999953688`
- Числовой: `24058`, `24084`, `0005802`
- Иногда отсутствует (строка без артикула)

**Вывод:** поле `sku` должно быть nullable; поиск по каталогу должен работать и по артикулу, и по наименованию.

**Наименования товаров:** длинные, включают объём (0,1 / 0,75 / 0,9 л / 12 кг), номер цвета, название цвета, бренд. Пример: «Колеровочная паста 0,9 л. №27 зеленый "ПалИж"».

**Вывод:** для карточки товара и фильтров желательны структурированные поля: `volume`, `colorNumber`, `colorName`. В MVP можно обойтись полнотекстовым поиском по `name`.

**Ценовые наблюдения:**

- Один и тот же товар (`9999956948`) встречается в заказе 201 по разным ценам: 1,00 и 163,67. Вероятно, акция / компенсация / коррекция. Платформа показывает цену из соглашения 1С (base + discount %); такие ручные корректировки — через менеджера.
- Разброс цен: от 37,64 (100 мл) до 3 671,14 (12 кг эмаль) и 2 090,00 (CS-980 премиум). Формат цены: два десятичных знака.

## 2. КП «Основы колористики»

### 2.1 Описание

Коммерческое предложение по курсу обучения для колористов. Организатор — Palizh (ООО «Новый дом»).

### 2.2 Форматы

| Формат | Длительность | 1 чел. | 2-3 чел. | 4-7 чел. |
| ------ | ------------ | ------ | -------- | -------- |
| Онлайн + офлайн | 2 дня | 25 000 | 42 000 | 68 000 |
| Офлайн | 2 дня | 44 000 | 72 000 | 100 000 |
| Офлайн (краткий) | 1 день | 24 000 | 41 000 | 67 000 |

### 2.3 Выводы для карточки курса на платформе

| Поле карточки | Источник из КП |
| ------------- | -------------- |
| Название | «Основы колористики» |
| Тип | Курс (стандартный) |
| Целевая аудитория | Колористы, операторы колеровочного оборудования, технологи |
| Формат | Онлайн + офлайн / Офлайн |
| Длительность | 1-2 дня |
| Стоимость | От 24 000 (зависит от формата и числа участников) |
| Программа | 15 пунктов (лекции + практика) |
| Место | г. Ижевск (офлайн) |
| Форма записи | ФИО, компания, кол-во участников, формат, комментарий |

### 2.4 Влияние на модель данных

- **Стоимость зависит от количества участников** → нельзя показать одну цену; нужна таблица цен или формат «от X руб.».
- **Несколько форматов одного курса** → одна карточка курса с выбором формата при записи, или отдельные карточки на каждый формат.
- **Форма записи** должна содержать поле «количество обучающихся» (подтверждает ЧТЗ 14).
