# Архив

Папка для устаревших артефактов корня проекта: отчёты аудитов прошлых дат, снимки документов, которые уже не актуальны, но должны быть доступны для истории.

**Что НЕ кладём в архив:**
- Снимки «Понимание задачи» по крупным изменениям — у них отдельная папка `Понимание задачи/архив/`.
- Транскрипты и саммари интервью — они остаются в `Интервью и встречи/` с датой в имени.

**Правило:** при появлении нового отчёта аудита в корне (`Аудит_документации_ГГГГ-ММ-ДД.md`) старые файлы аудита переезжают сюда.
