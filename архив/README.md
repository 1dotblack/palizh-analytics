# Архив

Папка для устаревших артефактов: отчёты аудитов прошлых дат, снимки документов, **устаревшие артефакты Инфарх**, которые уже не актуальны, но должны быть доступны для истории.

**Что НЕ кладём в архив:**
- Снимки «Понимание задачи» по крупным изменениям — у них отдельная папка `Понимание задачи/архив/`.
- Транскрипты и саммари интервью — они остаются в `Интервью и встречи/` с датой в имени.

**Правило:** при появлении нового отчёта аудита в **корне** репозитория (`Аудит_документации_ГГГГ-ММ-ДД.md`) предыдущий отчёт (включая последний в корне) переезжает в `архив/`. *На 2026-04 в корне отчёта нет — последний снимок: `Аудит_документации_2026-04-17.md` в этой папке.*
