# Записи интервью (исходные медиа)

Файлы **видео и аудио** из этой папки **не коммитятся в Git** (см. корневой `.gitignore`) — из‑за лимитов GitHub и размера репозитория.

Храните записи **локально** или в корпоративном облаке. Для проекта в репозитории используются **саммари** (`../Саммари/`) и при необходимости **транскрипты** (`../Транскрайб/`).
