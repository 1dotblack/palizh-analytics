# ЧТЗ: Поиск

**Статус:** драфт  
**Источники:** Понимание задачи, ЧТЗ 06 (витрина и каталог).  
**As-is / To-be:** as-is — витрины **нет**; клиент ищет товар через менеджера или по прайсу/предложению. to-be — поиск на витрине нового сайта (разделы 3–4).

---

## 1. Назначение

Описывает поиск товаров на платформе: синонимы и наименования от разных клиентов, исправление опечаток и транслита, поиск по описанию и свойствам, рекомендации и аналоги в выдаче. Интеграция с **Ensi Cloud Search** (основной выбор; SearchBooster не выбран). Цель — клиент быстро находит нужный товар по привычным названиям и атрибутам.

---

## 2. Термины (общие)

| Термин | Описание |
|--------|----------|
| Синонимы | Варианты наименований товара (как называют клиенты, артикулы, альтернативные названия) |
| Ensi Cloud Search | Выбранный поисковый сервис для каталога (по Пониманию задачи); SearchBooster рассматривался как альтернатива, но не выбран |

## 4. To-be: требования (драфт)

### 4.1 Функциональность

- Поиск по наименованию, артикулу, описанию и свойствам товара (данные из 1С/каталога).
- Исправление опечаток, транслита, неправильной раскладки; распознавание естественного языка — по возможностям выбранного поискового сервиса.
- **Синонимы и опечатки:** реализация — **либо на стороне 1С** (данные, номенклатура контрагента, выгрузка), **либо силами поисковой системы** (поддержка синонимов и опечаток). **Отдельный ручной список синонимов не ведётся.** В 1С есть сущность «номенклатура контрагента» (связь внутреннего названия/артикула с «чужим» названием у клиента). Набор атрибутов для расширенного поиска (назначение, материалы и т.п.) — согласовать с маркетингом (интервью 2026-03-04).
- Аналоги товаров и интуитивные рекомендации в выдаче — по возможностям сервиса и приоритету заказчика.

### 4.2 Интеграция

- Интеграция **Ensi Cloud Search**: индекс каталога (товары, атрибуты, синонимы), передача запроса, получение выдачи. Частота обновления индекса при изменении каталога в 1С — согласовать.
- Поиск должен корректно искать по описанию и свойствам — по Пониманию задачи.

### 4.3 Место в интерфейсе

- Поиск в шапке витрины (по Инфарх); результаты — переход в каталог/карточку товара. Подсказки (автодополнение) при вводе — опционально.
